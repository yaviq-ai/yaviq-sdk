# YAVIQ SDK

YAVIQ is a cost-optimization and cost-control platform for Large Language Model (LLM) usage.

This repository contains the **public SDKs, CLI tools, and documentation**
used by developers to integrate with the YAVIQ platform.

> ‚ö†Ô∏è This repository contains **client-side SDKs only**.  
> Core optimization engines, backend services, and infrastructure are maintained
> in private repositories and are not part of this codebase.

---

## What You Can Do with YAVIQ

- Reduce LLM token cost and inference cost
- Optimize prompts, RAG documents, and conversation history
- Estimate token savings before sending requests to LLMs
- Select optimization modes: safe, balanced, aggressive
- Monitor optimization metrics and usage telemetry
- Integrate via Node.js, Python, or CLI
